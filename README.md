# Lead Tracker MVP

**A simple, free lead management dashboard for small businesses.**

Deploy in 5 minutes. No installation needed for your team.

---

## 🚀 Quick Start

### Run Locally (30 seconds)

```bash
pip install -r requirements.txt
streamlit run app_python.py
```

Opens at `http://localhost:8501`

### Deploy Free (2 minutes)

1. Push this repo to GitHub
2. Go to https://streamlit.io/cloud
3. Deploy in Streamlit Cloud dashboard
4. **Live!** Share the URL with your team

See `GITHUB_DEPLOY.md` for detailed instructions.

---

## 📊 Features

✅ **Dashboard** — 4 key metrics + 2 charts  
✅ **All Leads** — Searchable database with filters  
✅ **Add Lead** — Quick 30-second form  
✅ **Analytics** — Score distribution, age analysis, source performance  
✅ **CSV Export** — Download anytime  
✅ **Persistent** — Data saves automatically (SQLite)

## 🎯 Workflow

1. **New lead comes in**
   - Click "Add Lead"
   - Fill form (30 sec)
   - System auto-scores & sets 2-day follow-up

2. **Track your pipeline**
   - View dashboard
   - See hot vs warm vs cold leads
   - Identify what's due today

3. **Optimize your marketing**
   - Check Analytics tab
   - See which sources bring best leads
   - Focus budget there

4. **Close deals**
   - Follow up on hot leads
   - Download CSV if you need
